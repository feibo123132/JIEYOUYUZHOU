# 宇宙星星互动App

一个浪漫的互动应用，让用户在虚拟宇宙中点亮专属星星，记录美好的互动时刻。  

## ✨ 功能特色

- 🌟 **3D星空背景**：使用Three.js打造的沉浸式宇宙体验
- 💫 **点亮星星**：输入昵称即可在星空中点亮专属星星
- 📱 **响应式设计**：完美适配移动端和桌面端
- 💾 **数据持久化**：支持本地存储和云端同步
- 🎨 **精美动画**：星星闪烁、光晕效果营造浪漫氛围

## 🚀 快速开始

### 安装依赖

```bash
pnpm install
```

### 启动开发服务器

```bash
pnpm run dev
```

### 构建生产版本

```bash
pnpm run build
```

## 🛠️ 技术栈

- **前端框架**：React 18 + TypeScript
- **3D渲染**：Three.js
- **状态管理**：Zustand
- **样式框架**：Tailwind CSS
- **后端服务**：Supabase（可选）
- **通知组件**：Sonner

## 📁 项目结构

```
src/
```

## 🎯 使用说明

1. **进入应用**：打开后会看到美丽的星空欢迎页面
2. **输入昵称**：在输入框中输入你的别称（2-12个字符）
3. **点亮星星**：点击"点亮星星"按钮，在随机位置生成你的专属星星
4. **查看详情**：点击任意星星可以查看点亮信息
5. **返回重玩**：点击返回按钮可以重新输入昵称

## 🔧 配置说明

### Supabase配置（可选）

如果需要使用云端数据存储，请配置以下环境变量：

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

如果不配置，应用会自动使用本地模拟数据库。
